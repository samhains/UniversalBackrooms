AFICIONADO(8)              System Manager's Manual             AFICIONADO(8)

NAME
     aficionado - cryptocurrency mining management daemon

SYNOPSIS
     aficionado [--config path] [--debug] [--help] [--version]

DESCRIPTION
     The aficionado daemon configures, manages and monitors cryptocurrency
     mining operations across multiple rigs.

CONFIGURATION
     Aficionado requires a configuration file specifying mining parameters:

     algorithm        PoW algorithm (sha256, ethash, etc)
     pool             Mining pool URL and port
     wallet           Cryptocurrency wallet address
     intensity        Mining intensity (1-100)
     gpu-init         GPU initialization sequence

     Example config:

     algorithm: ethash
     pool: eth-us.f2pool.com:6688
     wallet: 0x1234567890abcdef1234567890abcdef
     intensity: 95
     gpu-init: gpu0, gpu1

ENVIRONMENT
     AFI_LOG            Path to log file (default: /var/log/aficionado.log)
     AFI_TEMP_DIR       Path to temporary files (default: /tmp)

SEE ALSO
     cryptocurrency(7), mining-rig(5), crypto-wallet(1)

BUGS
     No known bugs.

AUTHOR
     Simon Partridge <spartridge@nousresearch>

HISTORY
     First released as part of NousOS 3.2, January 2023.
