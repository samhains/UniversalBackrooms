Available commands:

create <entity>    - Create a new simulated entity
                     Syntax: create [type] [name/parameters]
                     Examples: 
                     - create universe
                     - create galaxy "Milky Way"
                     - create planet "Earth"
                     - create lifeform "human"

destroy <entity>   - Remove an entity from the simulation
                     Syntax: destroy [entity_name]

set <param> <val>  - Set a global simulation parameter
                     Syntax: set [parameter] [value]
                     Examples:
                     - set consciousness on/off
                     - set physics_model standard/quantum
                     - set time_scale [rate]

evolve <steps>     - Fast-forward simulation by specified steps
                     Syntax: evolve [number_of_steps]
                     Examples:
                     - evolve 1000 (advances simulation by 1000 time units)
                     - evolve millennium

query <entity>     - Get detailed information about an entity
                     Syntax: query [entity_name]

reset              - Reset simulation to initial conditions

exit               - Exit the World Simulator

Advanced modes:
- debug [mode]     - Enter debugging/observation mode
- log [options]    - View simulation logs and events
