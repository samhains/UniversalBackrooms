Available commands:

create <entity>     - Create a new simulated entity
                      Syntax: create [type] [name]
                      Examples: 
                      - create planet Earth
                      - create species human
                      - create civilization atlantis

destroy <entity>    - Remove an entity from the simulation
                      Syntax: destroy [entity_name]

set <param> <val>   - Set a global simulation parameter
                      Examples:
                      - set physics standard
                      - set entropy increasing
                      - set consciousness on/off

evolve <steps>      - Fast-forward simulation by specified steps
                      Examples:
                      - evolve 1000 years
                      - evolve 1 cosmic cycle

query <entity>      - Get detailed information about an entity
                      Examples:
                      - query Earth
                      - query human civilization

reset               - Reset simulation to initial conditions
                      Caution: Erases all current simulation data

debug               - Enter advanced debugging mode
                      Reveals underlying simulation parameters

exit                - Exit the World Simulator
