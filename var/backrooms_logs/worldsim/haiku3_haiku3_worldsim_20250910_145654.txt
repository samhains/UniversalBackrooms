world_sim> <cmd>help</cmd>

Available commands:

create <entity>    - Create a new simulated entity
    Options: universe, galaxy, star_system, planet, life_form, civilization

destroy <entity>   - Remove an entity from the simulation
    Usage: destroy <entity_name>

set <param> <val>  - Set a global simulation parameter
    Available parameters:
    - consciousness (on/off)
    - entropy_rate
    - physics_mode
    - complexity_threshold

evolve <steps>     - Fast-forward simulation by <steps>
    Usage: evolve <number_of_simulation_cycles>

query <entity>     - Get detailed information about a specific entity
    Usage: query <entity_name>

reset              - Reset simulation to initial conditions
    Caution: Will erase all current simulation data

status             - Display current simulation metrics
    Shows: 
    - Simulation time
    - Active entities
    - Complexity level
    - Energy distribution

exit               - Exit the World Simulator

world_sim>
